Tentang

Profile tentang Universitas Indonesia Perpustakaan Universitas Indonesia

Tahun 2010 UI membangun sebuah gedung megah untuk perpustakaan. Pembangunan gedung ini mengacu kepada rencana strategi UI di mana salah satunya adalah integrasi di bidang sarana/fasilitas, sumber daya manusia, dan keuangan. Perpustakaan sebagai salah satu komponen fasilitas pemelajaran merupakan tempat di mana semua sivitas akademika UI dapat bertemu dan berinteraksi untuk mengembangkan ide-ide yang pada akhirnya akan menghasilkan kolaborasi penelitian dari berbagai subjek. Selain itu, tujuan integrasi ini juga adalah untuk efisiensi di berbagai aspek, seperti pengadaan koleksi dan pengelolaan SDM. Gedung bernama “The Crystal of Knowledge” ini selesai dibangun di awal tahun 2011, dan proses integrasi dimulai pada bulan Maret 2012. Adapun perpustakaan yang bergabung ke gedung baru tersebut adalah: Perpustakaan Pusat, Perpustakaan FIB, Perpustakaan FT, Perpustakaan FMIPA, Perpustakaan FIK, Perpustakaan FH Beberapa fakultas lain tetap memiliki perpustakaan di fakultas namun memindahkan sebagian koleksinya ke gedung baru, yakni: Perpustakaan FASILKOM, Perpustakaan Fakultas Psikologi, Perpustakaan FISIP, Perpustakaan FKM, dan Perpustakaan FEB. Dengan bergabungnya beberapa perpustakaan dari fakultas ke gedung baru, maka UI tidak lagi menggunakan istilah ‘perpustakaan pusat’ namun menjadi 'Perpustakaan UI'.

Kelebihan HTML5

HTML5 memiliki sejumlah kelebihan dibandingkan dengan HTML:

  • Elemen semantik seperti <header>, <nav>, <article>, <section>, dan <footer> memungkinkan markup yang lebih tepat.
  • Elemen <audio> dan <video> memungkinkan penyisipan multimedia tanpa plugin tambahan.
  • LocalStorage memungkinkan penyimpanan data lokal untuk menyimpan preferensi pengguna atau riwayat bacaan.
  • Elemen <canvas> dan dukungan SVG untuk visualisasi data dan grafis interaktif.
  • API geolocation untuk layanan berbasis lokasi.

Penerapannya dalam web Perpustakaan Indra mencakup penggunaan elemen <audio> dan <video> untuk koleksi multimedia, elemen semantik untuk struktur halaman yang lebih baik, penyimpanan lokal untuk data pengguna, dan elemen <canvas> atau SVG untuk visualisasi data.
