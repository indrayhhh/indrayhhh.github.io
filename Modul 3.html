Penjelasan Tipe CSS

1. Inline Style Sheet

CSS inline ditempatkan langsung pada elemen HTML menggunakan atribut style. Contoh penggunaan inline style pada elemen <p> untuk mengubah warna teks: 

          <p style="color: red;">Ini teks berwarna merah</p>

2. Internal Style Sheet

CSS internal ditulis di dalam tag <style> di dalam tag <head> HTML. Berikut contohnya dengan mengubah warna teks <h2>: 

          
              <head>
                  <style>
                      h2 {
                          color: green;
                      }
                  </style>
              </head>

3. Eksternal Style Sheet

CSS eksternal disimpan dalam file terpisah dengan ekstensi .css. File CSS ini kemudian dihubungkan ke halaman HTML menggunakan tag <link>. Contoh penggunaan di dalam file style.css: 

          
              /* style.css */
              body {
                  background-color: #f2f2f2;
              }
          
      
          
              <head>
                  <link rel="stylesheet" href="style.css">
              </head>

4. Paragraf Mengenai HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten dari halaman web. Dengan HTML, kita dapat menentukan judul, paragraf, tautan, gambar, dan elemen-elemen lainnya yang membentuk halaman web. HTML memberikan dasar untuk membangun halaman web dengan mengorganisir dan menyusun informasi dengan cara yang dapat diinterpretasi oleh browser.

5. Paragraf Mengenai CSS

CSS (Cascading Style Sheets) adalah bahasa gaya yang digunakan untuk mempercantik dan mengatur tampilan elemen-elemen HTML. Dengan CSS, kita dapat mengubah warna, ukuran, tata letak, dan gaya elemen, memberikan desain yang menarik dan konsisten pada halaman web. CSS bekerja secara bersamaan dengan HTML, memisahkan struktur dan tata letak dari presentasi visual sehingga memudahkan pemeliharaan dan perubahan desain.
